# プロダクト名
アルティメット将棋

## チーム名

チーム28 ゲーム好き一般人

## プロダクト説明


一般的な将棋のルールは同じですが、
・「成り（陣地に乗り込むと駒の種類が変わるルール）」は面倒なのでナシ
・敵の駒を倒してポイント獲得、ルール追加
・盤面をひっくり返す、超自由なゲーム体験
